## 結語 ##

Mongo DB 特性：

1. 分散式寫入
2. 分散式查詢
3. 有結構性的文件導向資料庫

TIP：在資料快滿時，要先增加節點。不然MongoDB會停在那裏處理資料平衡的問題。
